前言

返回 页首

    ERP系统是整个公司业务系统的核心。门店能否正常使用ERP系统直接影响到门店业务的开展。

    公司ERP系统是使用CITRIX客户端连接到服务器，对服务器实施远程操作。因此准确地说，门店电脑安装的是Citrix客户端而不是ERP。ERP实际上是一种约定俗成的说法。为了便于大家理解我们仍将其称做为“ERP客户端”。

ERP系统由电脑部和运维部共同维护，但不同的人员有不同的权限，并不是某一个人就能处理所有的ERP问题的，因此门店在遇到ERP的各种问题时，首先要弄清楚故障原因，再联系电脑部或者运维部的相关岗位处理，避免找不对人而耽误处理问题的时间。

安装（注意下载的“ERP.rar"必须先解压缩，然后再安装否则安装会出错）

返回 页首

        EPR客户端的安装这里提供手动安装和自动安装两种方法。推荐首先使用“自动安装”，只有在“自动安装”失败的情况才使用“手动安装”

    (1) 自动安装

自动安装的方法非常简单：直接运行ERP.exe即可实现自动安装。注意，在自动安装时，请不要再干预其安装。

  

  (2)手动安装

手动安装ERP客户端相对也是比较简单的，运行"DISK1\setup.exe "然后对弹出对话框点“NEXT”即可完成安装。

  

配置（新建连接）

返回 页首

在ERP客户端安装完成后，还需要添加服务器连接才可以登陆到公司的ERP服务器。

在添加ERP服务器连接之前，需要弄清一个概念“什么是ERP服务器连接”，从下图我们可以直观的理解什么是“什么是ERP服务器连接”

    添加“ERP服务器连接的步骤：

(1)  双击桌面的“Citrix Program Neighborhood”图标，打开Citrix 程序，双击“ADD ICA　connection”开始新建连接

　

（2）  “弹出的ADD ICA　connection”点”Next"

  

(3)  设置连接的名称和连接的服务器IP地址

  

服务器IP地址说明:

・     使用VPN登陆BBS和ERP的门店，服务器IP地址为：192.168.0.16

・     电讯的驻外人员，服务器IP地址为：192.168.0.19

・     珠宝门店及电讯的驻外人员，服务器IP地址为：192.168.0.117

・     请注意IP地址输入的格式，IP地址为四组数字，每组数字以点号隔开，中间不存在空格或者其他符号。因此在输入IP地址时，一定要注意不得有其他符号有空格，还要保证数字输入正确，否则不能登陆到ERP服务器。

（4）点“下一步”

  

（5）设置登陆服务器的用户名和密码

  

用户名和密码说明:

・     “192.168.0.16”、“192.168.0.19”服务器的用户名和密码均为：ZYDX

・     “192.168.0.117”服务器的用户名和密码均为：ZYZB

・     请注意：用户名和密码是区分大小写的，在设置是一定要加以区分。


（6）设置ERP连接窗口的大小

首先取消“User Default"前的“√”，然后将“windows size”里的值选择为” Percent of Screen Size" 并将“Percent”的值由“75”改为“100”。

  

（7） 后面的操作，只需要点一次“下一步”和“完成”即可完成。

ERP打印故障

ERP打印故障 ，是门店遇到最多最常见的故障。我们有专业针对此问题的教程，这里只说明一下导致打印故障年原因，故障排除方法就不再赘述。

导致打印故障的原因只有两个，一是打印机驱动安装不正确，二是登陆同一台ERP服务器的计算机名相同。

ERP登陆故障

故障现象一般只有两种：

1. “The citrix server you have selected is not accepting connections.（你选定了一个citrix服务器不能接受的连接。）”

2. There is no citrix server configured on the specified address.（指定的地址未在citrix服务器上配置）”。



故障原因有解决方法：

1. 使用VPN登陆的门店，没有登陆VPN。解决方法：请先登陆VPN。

2.     ERP客户端里的地址不一致。解决方法：检查ERP连接里的服务器IP地址设置。




・使用VPN登陆BBS和ERP的门店，服务器IP地址为：192.168.0.16

・电讯的驻外人员，服务器IP地址为：192.168.0.19

・珠宝门店及电讯的驻外人员，服务器IP地址为：192.168.0.117
